## 📌 Sobre o Projeto

Este projeto tem como objetivo estruturar e organizar processos operacionais e financeiros de uma empresa do setor técnico por meio de dashboards, modelagem de dados e automação de processos.

A proposta consiste em transformar um ambiente com controles manuais e descentralizados em um sistema estruturado, orientado por dados e indicadores estratégicos.

---

## 🎯 Problema

Empresas de pequeno e médio porte frequentemente operam sem processos formalizados, utilizando:

- Registros manuais
- Comunicação informal
- Planilhas desorganizadas
- Ausência de indicadores de desempenho

Isso gera falta de previsibilidade financeira, dificuldades no controle operacional e decisões baseadas em percepção, não em dados.

---

## 🚀 Objetivos

### Objetivo Geral

Estruturar dados operacionais e financeiros, transformando informações dispersas em inteligência gerencial.

### Objetivos Específicos

- Organizar base de dados operacional
- Criar modelo de dados para análise
- Desenvolver indicadores estratégicos e operacionais
- Automatizar processos repetitivos
- Criar base escalável para futura migração para banco de dados

---

## 🛠️ Tecnologias Utilizadas

- Power BI
- Excel
- Power Query
- Python
- Estrutura preparada para futura integração com banco de dados (SQL)

---

## 📊 Indicadores Desenvolvidos

Principais KPIs estruturados:

- Manutenções preventivas realizadas
- Equipamentos ativos
- Controle de entrada e saída
- Contas a pagar por vencimento
- Contas a receber por período
- Fluxo de caixa projetado

---

## 🧠 Metodologia

1. Levantamento de necessidades
2. Estruturação e organização dos dados
3. Modelagem para análise
4. Desenvolvimento de dashboards
5. Implementação de automações
6. Planejamento de escalabilidade futura

---

## 🔮 Evolução Futura

- Migração para banco de dados relacional
- Integração via API
- Desenvolvimento de sistema web
- Estruturação de pipeline de dados automatizado
